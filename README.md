## Overview
This repository explores research and frameworks for portfolio optimisation
in fintech, with a focus on fixed income instruments in India.

## Problem Statement
Bond portfolios are often constructed using static rules that ignore
macro conditions, credit risk trade-offs, and concentration effects.

## Approach
- Rule-based credit risk assessment for explainability
- Use of macroeconomic indicators as regime context
- Portfolio optimisation using quantitative techniques

## Status
Early-stage research. Design and documentation in progress.
